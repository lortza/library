== README

This is a media library app where a user can track and review their movies, books, games, and music albums. 

* Ruby version: 2.2.1

* Rails version: 4.2.3

* Database: postgresql

CURRENT STATUS:
* Movies, books, albums, and games are set up with polymorphic nested reviews.
* Devise users are set up but no relationships are established. Need to create join tables.
